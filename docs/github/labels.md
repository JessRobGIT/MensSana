# GitHub Labels — MensSana

## Typ-Labels

| Label | Beschreibung |
|---|---|
| `bug` | Etwas funktioniert nicht wie erwartet |
| `documentation` | Verbesserungen oder Ergänzungen der Dokumentation |
| `enhancement` | Verbesserung einer bestehenden Funktion |
| `feature` | Neue Funktionalität |
| `question` | Fragen zum Code oder zur Architektur |
| `ux` | UX- und Design-Feedback |

## Workflow-Labels

| Label | Beschreibung |
|---|---|
| `duplicate` | Dieses Issue existiert bereits |
| `good first issue` | Geeignet für Einsteiger |
| `help wanted` | Unterstützung erwünscht |
| `invalid` | Ungültiges oder irrelevantes Issue |
| `test wanted` | Tests fehlen oder müssen ergänzt werden |
| `wontfix` | Wird bewusst nicht umgesetzt |

## Prioritäts-Labels

| Label | Farbe | Beschreibung |
|---|---|---|
| `P1-critical` | Rot `#d73a4a` | Blockiert den Fortschritt, sofort angehen |
| `P2-high` | Orange `#e4790b` | Hohe Priorität, nächster Sprint |
| `P3-medium` | Gelb `#e4c200` | Mittlere Priorität, Planung |
| `P4-low` | Blau `#0075ca` | Geringe Priorität, Backlog |
| `P5-vision` | Lila `#7057ff` | Langfristige Vision, kein Zeitdruck |
