# MensSana

> Gespräch, Orientierung und Begleitung im Alltag

MensSana ist eine KI-gestützte Begleit-App für ältere Menschen — mit Fokus auf einfache Bedienung, natürliche Sprache und emotionale Kontinuität.

---

## Ziele

- Gespräche und Orientierung im Alltag erleichtern
- Medikamente, Termine und Erinnerungen strukturieren
- Caregiver und Familie sicher einbinden

---

## Projektstruktur

```
MensSana/
├─ docs/          # Dokumentation (GitHub, Architektur, Produkt)
├─ web/           # Frontend-Prototypen
├─ backend/       # Supabase-Schema, Policies, Functions
└─ .github/       # Issue-Templates, PR-Template
```
